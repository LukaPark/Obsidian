직업 테이블 스키마

Table Occupation {
   ID <pk>
   ParentID : 상위분류의 ID 를 참조 -> 최상위 분류일 경우 null
   Name: 직업 명
   ClassLevel: 분류의 레벨 <CategoryA | CategoryB | CategoryC>
   Description: 직업 설명
 }
   Index(parentID) : 인덱싱
